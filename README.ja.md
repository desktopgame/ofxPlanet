ofxPlanet
=====================================

ofxPlanetは地形生成ライブラリです。

Introduction
------------
ofxPlanetは地形生成ライブラリです。  
インスタンシングを使用して、平面だけで構成された世界を高速に描画します。  
* 2Dパーリンノイズで地形を生成できます。
* 3Dパーリンノイズで空洞を生成できます。
* 重み付けテーブルを使用して任意の構造物を生成できます。
